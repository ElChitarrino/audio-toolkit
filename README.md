# Audio Toolkit SE (audio-toolkit)

A quasar (Vue) project with npm and FastAPI backend. It enables music enthusiasts to analyze metadata and play around with any audio track, potentially retrieved via YT API.

## Idea
This simple tool shall provide audio analysis and metadata for a given video (preferred music) on Youtube.

A python backend with a simple React UI shall provide the user with the needed functionality.
The Youtube API will be used with a respective API key

## Install the dependencies
```bash
npm install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)



### Lint the files
```bash

```
